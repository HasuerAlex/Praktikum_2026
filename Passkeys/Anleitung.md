# Leitfaden: Einrichtung von Microsoft Authenticator & Passkey

## Teil 1: Microsoft Authenticator hinzufügen (2FA)

1. **Anmeldung:** Öffnen Sie den Browser und melden Sie sich auf [Office.com](https://www.office.com) an.
2. **Konto aufrufen:** Klicken Sie oben rechts auf Ihr Profilbild und wählen Sie **Konto anzeigen** aus.
3. **Sicherheitseinstellungen:** Navigieren Sie im linken Menü auf den Bereich **Sicherheitsinformationen**.
4. **Methode hinzufügen:** Klicken Sie auf die Schaltfläche **+ Anmeldemethode hinzufügen** und wählen Sie in der Dropdown-Liste **Authenticator-App** aus.
5. **Einrichtung abschließen:** Folgen Sie den Anweisungen auf dem Bildschirm, um die Authenticator-App zu verknüpfen.

## Teil 2: Passkey auf dem Smartphone einrichten

1. **App öffnen:** Öffnen Sie die **Microsoft Authenticator App** auf Ihrem Smartphone.
2. **Konto auswählen:** Tippen Sie auf Ihr hinterlegtes Konto.
3. **Passkey erstellen:** Wählen Sie den Punkt **Passkey einrichten** (oder *Passkey erstellen*).
4. **Anweisungen befolgen:** Folgen Sie den Schritten auf dem Display, um die Einrichtung mit Ihrer Smartphone-Sperre (z. B. Face ID, Fingerabdruck oder PIN) zu bestätigen.
